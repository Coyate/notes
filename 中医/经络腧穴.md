
##### 主要经络

- 经脉: 竖着的脉为经脉, 包括十二经和任督二脉
- 络脉: 横着的脉为络脉

穴位: 除了经络上的穴位, 还有经外奇穴(经验穴位), 阿是穴(按哪儿哪儿痛)


|     | 三阴         | 三阳         |
| --- | ------------ | ------------ |
| 手  | 手太阴肺经   | 手阳明大肠经 |
|     | 手厥阴心包经 | 手少阳三焦经 |
|     | 手少阴心经   | 手太阳小肠经 |
| 足  | 足太阴脾经   | 足阳明胃经   |
|     | 足厥阴肝经   | 足少阳胆经   |
|     | 足少阴肾经   | 足太阳膀胱经 |

-  阴阳互为表里, 脏主收藏, 腑主运化
-  经络一端为肢体末端, 一端为对应脏腑, 头为百阳之会, 阳经必经过头部
-  阴升阳降, 人的双手举起, 阴经的方向为脏腑到手部, 足部到脏腑
-  阳经的方向为手部到头再到脏腑, 脏腑到头部再到足部, 推拿时, 顺经为补, 逆经为泄
-  对于一个埋头耕地的人, 太阳可以晒到的地方为阳面, 晒不到的为阴面
-  阳面顺序为阳明少太阳, 阴面顺序为太阴厥少阴
