## 科目二

#### 调节座位

1. 关车门，先调整座位
2. 头一拳，手绷直，**掌跟**在方向盘上面
3. 后视镜**左右**调节至能**看到一半车身**, **上下**调节至轻微看见**后门把手**
4. 注意是否有车门未关闭, 再系安全带
5. 踩刹车，挂D档，拉下手刹


##### 侧方位停车

1. 驶入

    1. 进入先用雨刮器碰到黄线, 再左转打直
    2. 车头直的时候, **雨刮器点和右边黄线2-3指**时合适（和右边黄线30cm）
    3. 直角出现在后视镜上方时刹停
2. 倒车

    1. R档，松刹车，右后视镜黄色实线**直角消失**后**往右打1.5**
    2. 身子往左探出看左后视镜, 左后视镜**直角出现**方向盘**回正**
    3. 左后视镜**轮胎快碰线**时, 往**左打1.5**
    4. 和实线**平行时****刹停**
3. 开出

    1. 右手**打左转向灯**，左手**切D档**，松刹车
    2. **探身子看左后视镜直角**出现时**回正**, 要**慢**
    3. **雨刮碰黄色实线**方向盘**往右1圈**，雨刮点和右侧实线2－3指为正中
    4. 直角在**右后视镜上半**部分刹停


##### 倒车入库

1. 进入

    1. 方向盘正中对齐第二三根线中间，**偏第三根线起始**的位置
    2. 前进，在**手到肩膀之间**刹停
2. 右边倒车入库

    1. 倒车到左后视镜下沿和**黄线剩3－4指**，向**右打1.5**
    2. 观察右后视镜，看**车和直角**差多少距离，如果只有2-3指，先**回0.5圈**方向盘
    3. 当**后面车门把手**碰线时，观察两侧**那边窄**，看窄的后视镜，如果那边很歪，往反方向打方向盘
    4. 窄侧和黄线**平行时**，方向盘**回正**
    5. 左后视镜**下沿离黄线1指高**时, **刹停**
3. 开出到右边

    1. D档, 开出
    2. **前大灯位置和横实线**重合，向**右打1.5圈**
    3. 方向盘指向正前方时**回正**
    4. 虚线在手～肩之间, **刹停**
4. 开出到左边

    1. D档开出
    2. 前大灯位置和横实线重合，向**左打1.5圈**
    3. 方向盘指向正前方时回正
    4. 虚线在手～肩之间, 刹停
5. 左边倒车入库

    1. 倒车到左后视镜下沿和**黄线剩5－6指**，向**左打1.5圈**
    2. 观察左后视镜


##### S弯

1. 车头往右边进入，**靠右时再回正**
2. **前大灯碰撞到黄线**往**左打1圈**，大灯**贴着黄线**走
3. 大灯**碰到左侧黄线**时**回正**
4. **雨刷点碰到左边马路牙子**时往**右打1圈**
5. 保持能看见马路牙子, 擦着走
6. 车头向前时, **回正**


##### 直角转弯

1. 调整车头, 左大灯碰黄线, **打右转向灯**
2. 在**右门车把手碰着实线**时, 快速**向右打1.5圈**
3. 车头直的时候**回正**

## 科目三

### 进入

1. 准备好了？准备好才开始考试，绕车
2. 绕车：走一圈，按两个地方，等语音提示
3. 观察没有车在两个车道，打开门，进入
4. 调整座位
5. 调整后视镜能看见5分之1的车身，看见一半的前门把手
6. 检查灯光在不在初始位置！
7. 系安全带
8. 踩刹车，点火，开始模拟灯光考试


### 模拟灯光考试

1. 打开前照灯？
2. 近光：直行，跟车，会车，照明良好，有路灯
3. 远光：照明不良，没有路灯
4. 交替：近-远-近，夜间超车，急弯，坡道，拱桥，人行横道，没有信号灯的路口
5. 示廓，警报：临时停车，故障
6. 关闭所有灯光

### 驾驶

1. 起步

    1. 踩刹车，左转向灯，挂D档，拉手刹
    2. 拉完手刹10秒内要起步，看后视镜，安全了起步
2. 踩刹车

    1. 红绿灯口子一定要踩刹车
    2. 如果不是第一个排队，快到线了还是要踩一下刹车
    3. 学校，公交站前面要踩刹车
3. 停止：在有停标志的路口要停5秒才走
4. 转弯：打转向灯3秒后再转，连续变道需要快速补转向灯
5. 待行区：

    1. 直行待行：左转绿色，直行红色
    2. 左转待行：左转红色，直行绿色
6. 掉头：打满方向盘起步

    1. 如果左转道停止线是虚线，可以不等红绿灯直接掉头
7. 直行：直行基本不能动方向盘，可以刹停


### 停车

- 右转向灯，靠近车道30cm停车，用雨刮器对准右边后调整好停车
- 停车，挂P档，拉手刹，关转向灯，熄火
- 解开安全带，看后方安全，右手打开车门，出去看安全带有没有卡住车门，10s内关车门
